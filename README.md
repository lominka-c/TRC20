# How to use?

1. save the folder "TRC20" to your project
2. import module
  ```import TRC20```
3. initialize the user
   ```user1 = USER_WALLET(address="YOUR TRC20 ADDRESS")```
4. use the function. Example:
   ```user1.last_transactions()```
